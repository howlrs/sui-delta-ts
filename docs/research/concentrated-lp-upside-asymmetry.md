# 集中流動性LPのアップサイド非対称性分析

**Date**: 2026-03-03
**Status**: 確認済み — 運用判断に直接影響する重要知見

## 要旨

集中流動性 LP (CLMM) は、価格上昇時のアップサイドを大幅に制限する一方、下落時の損失はほぼ全額負う構造的な非対称性を持つ。狭いレンジ (±3%) で SUI/USDC LP を運用する場合、SUI 値上がり益の捕捉率は約40%にとどまり、HODLに対して大幅にアンダーパフォームする。

## 問題の本質: LP = カバードコール

集中流動性 LP は本質的に **カバードコール (covered call writing)** と同等のペイオフ構造を持つ。

- **得るもの**: 手数料収入（= オプションプレミアム）
- **失うもの**: レンジ上限を超える値上がり益（= コールが行使される）
- **負うもの**: 下落リスクのほぼ全額（= 原資産の下落）

## 定量分析: ±3% レンジの場合

### SUI +3% (レンジ上限到達)

$3,000 ポジション ($1,500 USDC + $1,500 SUI) の場合:

| 指標 | HODL | LP (±3%) |
|------|------|----------|
| 価値変化 | +$45 (+1.50%) | +$22 (+0.74%) |
| SUI残量 | 1,667 SUI (変わらず) | **0 SUI** (全売却) |
| 捕捉率 | 100% | **49%** |

LP は SUI が上がる過程で**自動的に全て売却**してしまう。これが IL の正体。

### SUI +10% (レンジアウト → リバランス → 繰り返し)

```
1ステップ (SUI +3%):
  LP利益:        +$22.17 (+0.74%)
  リバランスコスト: -$3.78 (swap fee 0.25% × ~50%)
  純利益:        +$18.39 (+0.61%)

SUI +10% = 約3.3ステップ:
  LP純利益:  $60.7
  HODL利益:  $150.0
  捕捉率:   40%
  損失:     $89.3 (= 逸失利益)
```

### SUI -10% (下落時)

```
LP損失:   ≈ $130 (IL増幅 + rebalance cost)
HODL損失:  $150
LP は損失の 87% を負う (さらにリバランスコストが加算)
```

### 非対称性まとめ

| SUI変動 | LP捕捉率 (対HODL) | 説明 |
|---------|-------------------|------|
| +3% | 49% | レンジ内で全SUI売却 |
| +10% | 40% | 複数回リバランス + コスト |
| +50% | ~35% | リバランスコスト累積 |
| -3% | 87% (損失負担) | レンジ内で全USDC → SUI化 |
| -10% | ~90% (損失負担) | IL増幅 + リバランスコスト |

**上昇の40%しか得られないが、下落の90%を負う。**

## レンジ幅による影響

| レンジ幅 | 上昇捕捉率 | 手数料効率 | リバランス頻度 |
|---------|-----------|-----------|--------------|
| ±1% | ~30% | 最高 | 非常に高い |
| ±3% | ~40% | 高い | 高い |
| ±8% | ~55% | 中程度 | 中程度 |
| ±15% | ~65% | 低い | 低い |
| Full range | ~75% | 最低 | なし |

狭いレンジほど手数料効率は良いが、アップサイド捕捉率はさらに悪化する。

## 運用への含意

### LP が有利な相場

- **横ばい〜低ボラ**: 価格がレンジ内に留まり、手数料が安定的に蓄積
- **レンジバウンド**: 上下しても一定範囲内 → 手数料最大化

### LP が不利な相場

- **トレンド相場 (上昇)**: 値上がり益の60%を逸失。HODLの方が大幅に有利
- **トレンド相場 (下落)**: IL + 下落損失でダブルパンチ。手数料で吸収不可
- **急変動 (いずれの方向も)**: レンジアウト → リバランスコスト発生

### SUI価格別の日次純利益推定 ($3K規模, ±3%レンジ)

| SUI動向 | 日次純利益 | 内訳 |
|---------|-----------|------|
| 横ばい | +$25〜35 | 手数料$40 - コスト$5-15 |
| 緩やか上昇 (+1%/week) | +$15〜20 | 手数料$40 - IL$15-20 - コスト$5 |
| 急上昇 (+5%/week) | +$5〜10 | 手数料$40 - IL$25-30 - コスト$5 |
| 下落 (-3%/week) | -$10〜-30 | 手数料$40 - IL$50-70 |

### $3K → $10K 到達シミュレーション

| シナリオ | 日次純利益 | 到達見込み |
|---------|-----------|-----------|
| SUI横ばい (LP最良) | ~$30 | 複利で ~6ヶ月 |
| SUI緩やか上昇 | ~$17 | ~10ヶ月 (HODLなら半分の期間) |
| SUI急上昇 | ~$7 | ~18ヶ月+ (HODLの方が圧倒的に速い) |
| SUI下落 | 赤字 | 到達困難 |

## 戦略的示唆

### 1. LP は「方向性に賭けない」ときの選択肢

SUI の大幅上昇を期待するなら、LP より HODL が合理的。LP は「方向がわからないが手数料で稼ぎたい」場合の戦略。

### 2. ポートフォリオ分割の検討

全額LPではなく一部をHODLに配分するハイブリッド戦略:

```
例: $3,000 のうち:
  $2,000 → LP (手数料獲得)
  $1,000 → SUI HODL (アップサイド確保)

SUI +50% の場合:
  LP部分: +$2,000 × 35% × 50% × 50% = +$175
  HODL部分: +$1,000 × 50% = +$500
  合計: +$675 (全額LP時の +$350 vs 全額HODL時の +$750)
```

### 3. レンジ幅のトレードオフを意識する

狭いレンジ (±3%) は手数料効率が高いが:
- アップサイド捕捉率が最も低い
- リバランス頻度が高い → コスト増
- トレンド相場で最もアンダーパフォーム

ボラティリティエンジンが広いレンジ (±8-15%) を推奨する場合、手数料効率は下がるがアップサイド捕捉率が改善する。

### 4. 規模別の合理性

| 規模 | LP合理性 | 理由 |
|------|---------|------|
| $1K-3K | 低い | 手数料収入が小さく、アップサイド逸失が相対的に大きい |
| $5K-10K | 中程度 | 手数料で生活費程度の収入、ヘッジとの組み合わせが可能に |
| $50K+ | 高い | 手数料収入が十分大きく、ポートフォリオの一部としてLP運用が合理的 |
